 Tkinter_Calculator

A simple desktop calculator application built using Python and the Tkinter library. The application performs basic arithmetic operations with a clean graphical user interface.

Technology Used

Python – Core programming language

Tkinter – GUI development

Features

Addition, Subtraction, Multiplication, Division

User-friendly graphical interface

Real-time input and result display

How to Run

Ensure Python is installed on your system.
